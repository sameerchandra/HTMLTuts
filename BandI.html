Let's talk about Block and inline elements in HTML

Block Elements

Block Elements occupy the entire width available to it

Block Elements always start on a new line

Block Elements always have a pre set margin on them

Examples of block elements

    
        <div>
        <p>
        <ul>
        <ol>
    
    All header tags

<div>

  • The div tag is used as a container to segregate blocks of elements
  • When used with CSS, we can style blocks of elements instead of styling each element individually

This and the next list are in the same div

  • This is in the same div as the above paragraph
  • Let's style this div

Inline Elements

Inline Elements do not start on a new line

It only takes up width as necessary

Examples of inline elements

    
        <span>
        <img>
        <button>

<span>

  • Span is an inline element used when you want to style parts of a text
  • Generaly used together with CSS
